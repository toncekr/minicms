# Weedpal

Weedpal je jednoduchá sociální aplikace pro záznam a hodnocení odrůd konopí. 

Uživatelé se mohou zaregistrovat, přidat fotku, ohodnotit vlastnosti konkrétní odrůdy, zapsat si k ní poznámky a prohlížet si příspěvky od ostatních uživatelů ve veřejném feedu.

## Technologie

- **Frontend & Backend**: Next.js 16 (App Router), React 19
- **Stylování**: Tailwind CSS v4
- **Databáze**: PostgreSQL (přes Prisma ORM)
- **Autentizace**: Auth.js v5
- **Úložiště fotek**: Vercel Blob
